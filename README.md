# JavaScript Color Changer

A simple web application that changes the background color and displays a random motivational message when the button is clicked.

## :sparkles: Features

- Random background color
- Random text message
- Dynamic UI updates
- Smooth transition effects
- Interactive button

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## :books: What I Learned

- DOM Manipulation
- addEventListener()
- Functions
- Arrays
- Math.random()
- Math.floor()
- Dynamic CSS with JavaScript

## :rocket: Future Improvements

- Copy RGB color to clipboard
- Show HEX color
- Dark/Light mode
- Save color history
